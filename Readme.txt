System Requirement:
-Python 3.10 or latest
-PyYaml
-Jinja2
-Paramiko
-Json

Langkah-langkah untuk menggunakan Framework ini:

Instal kebutuhan sistem pada komputer lokal Anda.
Tambahkan alamat IP, username, dan password pada file devices.json.
Edit file YAML sesuai dengan alokasi setiap lokasi yang akan dikonfigurasi.
Jalankan mainoop.py.
Selesai, Anda akan melihat hasilnya.
Verifikasi konfigurasi:
Dapatkan daftar tunnel IPsec.
Dapatkan informasi router dengan ringkasan BGP.
Jalankan diagnose sys sdwan service atau diagnose sys virtual-wan-link service.
